# 项目简介

## 核心功能
- 基于YOLOv8的苹果成熟度检测，支持识别不成熟、成熟和中等成熟三类苹果
- 提供图片、视频及摄像头三种输入方式的检测功能
- 支持自定义置信度和IOU阈值
- 提供UI界面实时显示目标位置、目标总数和置信度信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能
- 支持选择不同检测模型
- 包含1600张标注数据集和训练好的模型

## 适用场景
- 果园苹果成熟度评估
- 农产品质量分级
- 农业自动化检测系统
- 计算机视觉教学与研究
- 智能农业监控系统

## 技术特点
- 采用Python 3.9开发，依赖opencv-python、PyQt5、ultralytics和torch1.9等库
- 提供完整的程序文件、UI界面源文件和数据集
- 包含项目运行教程，环境配置后可直接使用
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
